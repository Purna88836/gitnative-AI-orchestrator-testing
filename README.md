# gitnative-AI-orchestrator-testing

This repository has two separate surfaces:

- **Orchestrator scaffold** for issue-driven agent execution. Start with [SETUP.md](./SETUP.md).
- **Todo demo workload** used to exercise the scaffold with a small full-stack app. Start with [demo/todo-app/README.md](./demo/todo-app/README.md).

Use the scaffold docs when configuring automation for a repository. Use the demo docs when you want to run, test, or inspect the Todo example itself.
